### Interaction

Passing an onClick handler to the buttons will let you receive mouse clicks. The event is not a native browser event.
It's a <a href="https://reactjs.org/docs/events.html" target="_blank">SyntheticEvent</a> from React.

### Styling

You can customize the styling of a button by passing in a `className`.
The button component has two unused class names applied to the container and the label.
These are `button-container` and `button-label`.

```css
.tomato-primary.button-container {
  background-color: tomato;
}

.white-label.button-label {
  background-color: white;
  color: tomato;
  font-weight: bold;
  padding: 4px;
}
```
