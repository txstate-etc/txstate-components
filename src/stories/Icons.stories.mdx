## Props
Fill and stroke colors may do different things for icons, depending on how they're drawn. Setting fill
to a specific color will _usually_ color the entire icon. In some cases stroke will need to be set as well.

| prop          	|    type   	|
|-----------------|:-----------:|
| width           | number   	  |
| height       	  | number 	    |
| stroke        	| string     	|
| fill 	          | string     	|
