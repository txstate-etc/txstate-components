## Interaction

Passing an onClick handler to the buttons will let you receive mouse clicks. The event is not a native browser event. 
It's a <a href="https://reactjs.org/docs/events.html" target="_blank">SyntheticEvent</a> from React.

## Styling

You can customize the styling of a button by passing in a `className`. 
The button has a spare class name `txst-button` applied to it. It also uses the variant names as
class names: `primary`, `outline`, and `transparent`.

```css
.tomato-primary.txst-button.primary {
  background-color: papayawhip;
  border-color: tomato;
}

.tomato-primary:focus {
  box-shadow: 0px 0px 10px -2px tomato;
  outline: none;
}

.white-label.txst-button.primary {
  color: tomato;
}
```
