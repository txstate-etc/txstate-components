# Toast
---

```js
import { Toast } from 'txstate-react/lib/components/Toast'
```

## Arguments

The gravity positions listed below are available when importing the `Toast` component under `Toast.GRAVITY`

| Name     | Description                                                                                                                                             | Default    |
|----------|---------------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| message  | **Text to display in the Toast** <br /> string | -- |
| duration | **Length to display the toast, in ms** <br /> number | 3000 |
| gravity  | **Position of the toast** <br /> GRAVITY.TOP_LEFT \| GRAVITY.TOP_CENTER \| GRAVITY.TOP_RIGHT \| GRAVITY.BOTTOM_LEFT \| GRAVITY.BOTTOM_CENTER \| GRAVITY.BOTTOM_RIGHT | TOP_CENTER |
